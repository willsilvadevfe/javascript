Criar uma classe ContaBancaria que tenha uma propriedade privada para saldo. Utilizar getters e setters para acessar e atualizar o saldo. O saldo só pode ser utilizado através de depósitos e saques, e não deve permitir que o saldo se torne negativo. Implementar métodos para depósito (depositar(valor)) e saque (sacar(valor)), onde o saque só é permitido se houver saldo suficiente.
