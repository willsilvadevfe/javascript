Criar uma classe produto que inicie sem itens no estoque. A classe deve ter métodos públicos para remover itens do estoque (comprar(quantidade)) e para adicionar itens no estoque (AdicionarAoEstoque(quantidade)). Além disso, implementar um método privado #removerDoEstoque(quantidade) que será usado internamente para atualizar a quantidade do estoque sempre que comprar() for efetuado com sucesso. O acesso à quantidade em estoque deve ser restrito.
